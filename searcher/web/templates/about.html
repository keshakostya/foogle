About page

Search engine

This is a search engine, thats searches local files by its content. You can search with queries with logical operators. You can save and load index, that was built.

Technology

This search engine is written in Python. For web interface it uses Flask (NO Javascript, pure Python). Engine searches with Tf-Idf metric(https://en.wikipedia.org/wiki/Tf%E2%80%93idf) to score documents.
